**用于团队初始开发时统一编码规范**

主要有如下功能点

- 针对不同编辑器生效
- vocde 在项目打开时会有提示安装推荐插件（ 在你没有安装 eslint，vetur 等 ）
- 对 vscode 编辑器配置做团队统一规范，对诸多忽略文件（夹）做了处理
- 对 git 前端项目做了诸多忽略文件（夹）处理
- 对 eslint 做了诸多忽略文件（夹）处理
- 对 prettier 做了诸多忽略文件（夹）处理
- 兼容了 eslint 与 prettier 编码规范
